# moonlauncher
MoonLauncher — быстрый Minecraft-лаунчер с открытым исходным кодом. Играйте в любые версии, от классики до новейших снапшотов, используя локальные (пиратские) . Главные плюсы: минимальное потребление ресурсов, полная прозрачность кода без скрытых угроз.Настраивайте игру под себя и запускайте в один клик!
