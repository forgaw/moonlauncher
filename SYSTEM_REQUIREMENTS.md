# Системные требования / System Requirements

## RU
### Moonlauncher (приложение)
- ОС: Windows 10 (22H2+) или Windows 11, x64.
- CPU: 2+ ядра, 2.0 GHz и выше.
- ОЗУ: минимум 4 GB (рекомендуется 8 GB+).
- Диск: от 2 GB для лаунчера и служебных файлов.
- Интернет: стабильное подключение для загрузки версий, библиотек, новостей и контента.
- WebView2 Runtime: Microsoft Edge WebView2 Runtime (Evergreen).
- Права: обычный пользователь (администратор нужен только для MSI-установки/обновления).

### Minecraft через Moonlauncher
- Java:
  - Java 8 для старых версий (1.16 и ниже),
  - Java 17 для 1.17–1.20.4,
  - Java 21 для новых версий/сборок (при необходимости).
- ОЗУ для игры:
  - Vanilla: 2–4 GB,
  - Modded/сборки: 4–8 GB+.
- Диск под игру (`%USERPROFILE%\\MoonMine` по умолчанию):
  - Vanilla: от 5 GB,
  - С модами/ресурсами/шейдерами: от 20 GB (рекомендуется 40 GB+).
- GPU: поддержка OpenGL 3.2+ (минимум), рекомендуется дискретная видеокарта для шейдеров.

### Сеть и источники
- Новости: используется несколько источников (официальные + региональные fallback).
- Контент (моды/пакеты/шейдеры): требуется доступ к API Modrinth/CurseForge/RF.

## EN
### Moonlauncher (application)
- OS: Windows 10 (22H2+) or Windows 11, x64.
- CPU: 2+ cores, 2.0 GHz or higher.
- RAM: 4 GB minimum (8 GB+ recommended).
- Disk: at least 2 GB for launcher/runtime files.
- Network: stable internet for versions, libraries, news, and content downloads.
- WebView2 Runtime: Microsoft Edge WebView2 Runtime (Evergreen).
- Permissions: standard user is enough (admin rights are needed for MSI install/update only).

### Minecraft via Moonlauncher
- Java:
  - Java 8 for older versions (1.16 and below),
  - Java 17 for 1.17–1.20.4,
  - Java 21 for newer versions/modpacks (when required).
- RAM allocation:
  - Vanilla: 2–4 GB,
  - Modded/modpacks: 4–8 GB+.
- Game storage (`%USERPROFILE%\\MoonMine` by default):
  - Vanilla: 5 GB+,
  - With mods/resource packs/shaders: 20 GB+ (40 GB+ recommended).
- GPU: OpenGL 3.2+ minimum; discrete GPU recommended for shaders.

### Network and sources
- News uses multiple sources (official + regional fallbacks).
- Content (mods/packs/shaders) requires access to Modrinth/CurseForge/RF APIs.
